# Plotting

This project covers the basics of data visualization using `numpy` and `matplotlib`.

## Tasks

### 0. Line Graph
Plot $y = x^3$ as a solid red line with an x-axis range from 0 to 10.

### 1. Scatter
Plot Men's Height vs Weight as a scatter plot with magenta points.

### 2. Change of scale
Plot exponential decay of C-14 with a logarithmic y-axis and specific x-axis range.

### 3. Two is better than one
Plot multiple exponential decay lines (C-14 and Ra-226) with custom styles and a legend.

### 4. Frequency
Plot a histogram of student scores with custom bins and outlines.

### 5. All in One
Combine all previous plots into a single figure using a 3x2 grid layout.

### 6. Stacking Bars
Plot a stacked bar graph representing fruit quantities per person with custom colors.
